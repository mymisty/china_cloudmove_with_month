# DATA_SOURCES

本文件记录“中国雨带的季节移动”页面采用的资料依据。页面采用的是课堂教学概括表达，非逐日预报；月份和区域以权威科普资料、业务气候平均描述和公开论文中的“常年/气候态”表述为基准。

## 关键校正

- 用户初始建议包含“3—4月：华南前汛期开始”。资料显示华南前汛期通常指 4—6 月，中国天气网称华南每年受夏季风影响最早为“4月前后”，中国气象局科普称常年开始时间为 4月6日。因此页面最终写为“4月前后：华南前汛期启动”，未把 3 月作为主雨带阶段。
- 5 月节点没有写成独立稳定雨带，而写成“江南多雨过渡、北进中的摆动”，因为资料更多支持 5 月前后雨季自南向北开启、6 月中旬后梅雨区明显建立。
- 9—10 月节点写成“南撤、结束”，并保留“华南沿海仍可能受台风或热带系统影响”，避免误导为全国完全无雨。

## 气象资料来源

### CMA_STORM_FAMILY

- 资料来源名称：中国气象局《盘点暴雨家族》；对应长标题为中国气象局《盘点“暴雨家族”：不同个性的它，你了解吗？》
- 网址：https://www.cma.gov.cn/2011xzt/2021zt/20210716/2014072105/202107/t20210716_580825.html
- 采用内容：锋面雨带由夏季风主导并随副热带高压作季节性移动；4—6 月暴雨区主要在华南，6—7 月长江中下游出现梅雨，7月中旬至8月下旬主雨带北移至华北、东北。
- 对应页面节点：sc-pre-flood、jianghuai-meiyu、north-northeast-rains。
- 表达方式：页面中的“北进、停滞、北跳”是概括表达；阶段月份和区域来自资料整理。

### CMA_CHINA_PRECIPITATION

- 资料来源名称：中国气象局《中国的降水》及华南前汛期科普资料
- 网址：https://www.cma.gov.cn/2011xzt/2013zhuant/20130326/2013032601/201303/t20130328_209279.html
- 采用内容：华南 4—6 月为第一个多雨季节，冷暖空气作用和季风爆发是主要原因；7—9 月华南后汛期多受台风等热带天气系统影响。
- 对应页面节点：sc-pre-flood、rainy-season-end。
- 表达方式：页面“华南沿海仍可能受台风或热带系统影响”为依据资料整理。

### CMA_SUMMER_MONSOON_RAINSTORM

- 资料来源名称：中国气象局《夏季风与我国暴雨》相关科普脉络
- 网址：https://www.cma.gov.cn/2011xzt/2021zt/20210716/2014072105/202107/t20210716_580825.html
- 采用内容：夏季风暖湿气流北上，与冷空气交汇形成锋面雨带；雨带随季风和副热带高压位置发生季节移动。
- 对应页面节点：jiangnan-rains、north-northeast-rains、south-retreat、rainy-season-end。
- 表达方式：页面“东南水汽输送”“暖湿气流北上”为教学化概括。

### CMA_SUBTROPICAL_HIGH

- 资料来源名称：中国气象局《图解副热带高压》；并参考中国气象局公共科普中关于副热带高压北抬与雨带阶段性移动的描述
- 网址：https://www.cma.gov.cn/2011xwzx/2011xmtjj/202507/t20250708_7198365.html
- 采用内容：中国气象局媒体转载资料提到我国雨带有阶段性南北进退规律：4—6 月华南前汛期，6—7 月江淮梅雨，7—8 月华北和东北雨季；北方湿热与副热带高压北抬有关。
- 对应页面节点：jiangnan-rains、jianghuai-meiyu。
- 表达方式：页面“副热带高压北抬、北跳”是依据资料整理的简化表述。

### WEATHER_REGIONAL_RAINSTORM

- 资料来源名称：中国天气网《我国区域性暴雨概况》
- 网址：https://www.weather.com.cn/science/lssj/05/565622.shtml
- 采用内容：华南每年受夏季风影响最早（4月前后）、结束最晚（10月前后），前汛期为4—6月，后汛期为7—9月；9—10月夏季风向南撤退，大陆地区降雨明显减少。
- 对应页面节点：sc-pre-flood、jianghuai-meiyu、north-northeast-rains、south-retreat、rainy-season-end。
- 表达方式：页面 4 月前后和 10 月前后的节点直接依据该资料；“大陆大部退出主雨季”为概括表达。

### WEATHER_RAIN_SEASON_DISTRIBUTION

- 资料来源名称：中国天气网《全国雨季进程图》；并参考中国天气网华南前汛期资料与全国雨季进程类报道
- 网址：https://www.weather.com.cn/life/2020/03/3303529.shtml
- 采用内容：华南前汛期是我国华南地区 4—6 月间出现的多雨时期，区别于 7—10 月以台风、热带辐合带等热带系统影响为主的后汛期。
- 对应页面节点：sc-pre-flood、rainy-season-end。
- 表达方式：页面“前汛期/后汛期”和“台风或热带系统”来自资料整理。

### WEATHER_RAIN_IMPACT_CHINA

- 资料来源名称：中国天气网《一场雨如何影响中国？》；对应长标题为中国天气网《雨如何影响中国？一条神奇的线见证中国历史变迁》
- 网址：https://www.weather.com.cn/zt/tqzt/3641555.shtml
- 采用内容：每年 5 月前后，随着夏季风到来，我国中东部雨季自南向北开启；华南龙舟水、江南梅雨、“七下八上”的华北暴雨是典型节点。
- 对应页面节点：jiangnan-rains。
- 表达方式：页面“雨季自南向北开启”依据该资料，江南阶段为教学化过渡节点。

### RAINY_SEASONS_REVIEW

- 资料来源名称：《应用气象学报》汛期我国主要雨季进程成因及预测应用进展
- 网址：https://html.rhhz.net/yyqxxb/html/20190401.htm
- 采用内容：东亚夏季风北推和南撤使华南、长江流域、华北等相继进入降水集中期；华南前汛期、梅雨、华北雨季是我国主要雨季进程；文中气候态采用 1981—2010 年平均。
- 对应页面节点：sc-pre-flood、jianghuai-meiyu、north-northeast-rains。
- 表达方式：页面阶段顺序和“降水集中期”来自资料整理；具体 UI 文案为概括。

### EASM_RETREAT_REVIEW

- 资料来源名称：《大气科学》论东亚夏季风的特征、驱动力与年代际变化
- 网址：https://www.dqkxqk.ac.cn/dqkx/dqkx/ch/html/20180307.htm
- 采用内容：东亚夏季风季节进退控制中国东部夏季降水带的阶段性变化；长江中下游梅雨区和华北、东北夏季降水具有不同季节特征。
- 对应页面节点：south-retreat、rainy-season-end。
- 表达方式：页面“夏季风减弱、雨带南撤”为概括表达，并由中国天气网 9—10 月南撤描述共同支撑。

## 地图底图来源

### NASA_BLUE_MARBLE_JULY

- 资料来源名称：NASA Blue Marble Next Generation July 2004 base map
- 网址：https://assets.science.nasa.gov/content/dam/science/esd/eo/images/bmng/bmng-base/july/world.200407.3x5400x2700.jpg
- 采用内容：NASA 真实色全球地球底图。下载后裁剪出东亚/中国区域，保存为 `public/images/nasa-china-july-crop.jpg`；同时裁剪出华南、长江中下游、华北东北三张区域影像参考窗口。
- 对应页面节点：`RainBeltMap.jsx` 的真实卫星底图与三张区域影像参考窗口。
- 表达方式：NASA 影像用于真实地表纹理和课堂视觉可信度；雨带云团、路径和降水范围仍是教学概括动画，不是逐日卫星云图或业务级降水产品。

### DATAV_GEOATLAS_CHINA

- 资料来源名称：阿里云 DataV.GeoAtlas 中国省级行政区划 GeoJSON
- 网址：https://geo.datav.aliyun.com/areas_v3/bound/100000_full.json
- 采用内容：中国省级行政区划边界，包括省、自治区、直辖市、台湾省、香港特别行政区、澳门特别行政区。
- 对应页面节点：`RainBeltMap.jsx` 中的真实行政区划底图；本地文件为 `public/data/china-admin-geo.json`。
- 表达方式：底图用于课堂位置识别，不用于精确 GIS 测绘；页面雨带仍是教学概括层。
