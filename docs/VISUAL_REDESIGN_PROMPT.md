# VISUAL_REDESIGN_PROMPT

## 目标

重做“中国雨带季节移动”页面的核心视觉层，使它更像真实地理教学演示，而不是廉价抽象 UI。

## 设计提示词

使用 NASA 真实地球卫星影像作为底图，叠加中国省级行政区划边界，整体像一张课堂投屏用的气象地理演示图。地图底色应有真实陆地、海洋、山地纹理，而不是手绘渐变。雨带不再是单一廉价椭圆，而是由多层半透明降水云团、雨幕粒子、边缘雾化和局部强降水核心组成。雨带出现、增强、停滞、北跳、南撤、消散时，配套的三张气象意象图片需要随阶段淡入淡出：华南暖湿水汽、江淮梅雨云团、北方暴雨云系。图片不能像卡片堆砌，要作为地图旁的“气象窗口”或浮层，与当前雨带节奏同步。

水汽输送箭头不能是统一拉直的箭头，应改成分段、弯曲、沿路径流动的水汽流线。每段路径对应一个阶段：华南起步、江南北推、江淮停滞、华北东北北跳、南撤回落、退出收束。箭头需要有输送感：虚线粒子沿曲线移动、箭头头部随段落出现，北进和南撤颜色不同，停滞阶段用旋涡/环形流线表达蓄积。雨带移动之间要有过渡：上一阶段云团残影逐渐淡出，下一阶段雨带先出现弱云团再增强，避免硬切。

整体 UI 保持教学用途：少文字、强地图、强时序、适合 16:9 投屏。不要添加营销页或装饰性大卡片。所有真实数据、图像来源写入资料文档，页面运行时使用本地资源，不依赖外网。

## 本轮执行清单

1. 检索并采用 NASA 真实卫星底图，保存为本地静态资源。
2. 保留省级行政区划边界作为定位层。
3. 重做雨带视觉：多层云雨、粒子、阶段增强/消散。
4. 添加三张气象图片/气象窗口，并随阶段出现或消失。
5. 将水汽输送箭头改为分段曲线路径和流动粒子。
6. 路径分段，不再使用统一拉直箭头。
7. 更新资料来源、长期记忆和核查脚本。
